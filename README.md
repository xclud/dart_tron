Dart API Library for interacting with the TRON Network.

[![pub package](https://img.shields.io/pub/v/tron.svg)](https://pub.dartlang.org/packages/tron)

## Getting started

In your `pubspec.yaml` file add:

```dart
dependencies:
  tron: any
```

## Usage

```dart
import 'package:tron/tron.dart';
```
